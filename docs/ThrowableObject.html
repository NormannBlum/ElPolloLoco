Class: ThrowableObject

ThrowableObject(x, y, direction, world)

Represents a throwable object (e.g., a bottle) that rotates in the air and triggers a splash animation upon collision. Inherits from `MovableObject` and includes both rotation and splash animations.

Constructor

new ThrowableObject(x, y, direction, world)

Creates an instance of a throwable object.

direction number The throw direction (1 = right, -1 = left).
world Object The world in which the object exists.
