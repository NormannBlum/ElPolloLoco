Class: Level

Level(enemies, clouds, backgroundObjects, bottles, coins)

The Level class represents a level in the game. It contains enemies, background objects, collectible items, and the end position of the level.

Constructor

new Level(enemies, clouds, backgroundObjects, bottles, coins)

Creates an instance of a level with enemies, clouds, background objects, bottles, and coins.
