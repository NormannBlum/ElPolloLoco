Class: BackgroundObject

BackgroundObject(imagePath, x)

The BackgroundObject class represents a movable background object in the game. It extends the MovableObject class and moves along with the game world.

Constructor

new BackgroundObject(imagePath, x)

Creates an instance of a background object.
Parameters:
Name Type Description
imagePath string The path to the background object's image.
x number The x-position of the background object.
